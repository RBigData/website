This and subsequent versions as a single gzipped tar file are available [here](https://pbdr.org/releases/). There are two sets of instructions: one for those who wish to use Docker, and one for those who do not. Under most circumstances, using the Docker builds is much easier. This is a good fit for anyone wishing to use pbdR: * On a desktop/laptop (caveat: OS version has to be reasonably new) * In the cloud. * On a cluster running Singularity.
